# SCHEDULIX — Timetable Manager 🗓️

A modern desktop application built to handle academic scheduling smoothly and efficiently. The app securely separates access for different users and communicates directly with a database to keep schedules updated in real time.

* **Repository Link:** [https://github.com/yuvarajsgit/Timetable-Manager](https://github.com/yuvarajsgit/Timetable-Manager)

##  Core Features

* **3-in-1 User Portals:** The system automatically creates separate, secure areas for **Administrators**, **Faculty Members**, and **Students**. Each user type gets their own custom access panel, keeping data private and organized.
* **Instant Schedule Updates:** When a user navigates between different dashboard options, the app tracks exactly which management tool or calendar view is active and automatically refreshes the main view area instantly.
* **Click-to-Refresh Actions:** Behind the scenes, the moment an action button is clicked, the app automatically triggers a background data request to update the table contents on your screen.
* **Live Database Connection:** The scheduling engine links directly to a secure SQL database server. This allows the program to safely fetch up-to-date timetable charts, teacher workloads, class codes, and room assignments in real time.
